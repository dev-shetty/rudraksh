## Technologies Behind Rudraksh

*Rudraksh leverages a robust stack of modern technologies to deliver a seamless experience for its users and maintain high accuracy in disease prediction.*

### Disease Prediction & Model Training:

- **[Python](https://www.python.org)**: The core language used for disease prediction, offering flexibility and efficiency.
- **[Google Colab](https://colab.research.google.com)**: A powerful tool used for training our machine learning models in the cloud.
- **[TensorFlow](https://www.tensorflow.org)**: An open-source machine learning framework that aids in building and training the prediction models.
- **[Numpy](https://numpy.org)**: Essential for numerical computations in our prediction algorithms.
- **[Flask](https://flask.palletsprojects.com/en/2.3.x)**: A lightweight web framework that integrates our Python-based algorithms with our web frontend.
- **[PyCharm IDE](https://www.jetbrains.com/pycharm)**: Our preferred integrated development environment for Python development, offering robust tools and features.

### Web Development & Deployment:

- **[Next.js](https://nextjs.org/docs)**: A React framework that enables server-rendered React applications, enhancing performance and user experience.
- **[Typescript](https://www.typescriptlang.org)**: A typed superset of JavaScript ensuring robustness and better code quality.
- **[TailwindCSS](https://tailwindcss.com)**: A utility-first CSS framework used for crafting our web interface.
- **[Shadcn UI](https://ui.shadcn.com)**: Shadcn UI is a free and open-source React UI library that provides beautiful and accessible components that are easy to use and customize.
- **[Git and GitHub](https://github.com)**: Essential tools for version control, collaborative development, and code reviews.
- **[Vercel](https://vercel.com)**: Our preferred platform for deploying and hosting the Rudraksh web application, offering fast, global deployment.
- **[ESLint](https://eslint.org)**: Ensures code quality and consistency by detecting and fixing problematic patterns in the codebase.
- **[npm](https://www.npmjs.com)**: The package manager used for managing and installing project dependencies.
